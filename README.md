# ChurnShield

A machine learning-powered REST API that predicts customer churn for telecom companies. Built with FastAPI and scikit-learn, trained on the Telco Customer Churn dataset, and deployable via Docker or Render.

## Overview

ChurnShield accepts customer feature data via a POST request, preprocesses it using a saved column schema, and returns a churn prediction with a probability score and risk level.

**Risk levels:**
-  `high` — probability > 0.7
-  `medium` — probability 0.4–0.7
-  `low` — probability < 0.4

The model was trained on the [IBM Telco Customer Churn dataset](https://www.kaggle.com/datasets/blastchar/telco-customer-churn) and serialized as `best_model.pkl`.

## Getting Started

### Prerequisites

- Python 3.11+
- PostgreSQL (optional — only needed for `load_data.py`)

### 1. Clone the repo

```bash
git clone https://github.com/hiba-khan/churnshield.git
cd churnshield
```

### 2. Create and activate a virtual environment

```bash
python -m venv venv
# Windows
venv\Scripts\activate
# macOS/Linux
source venv/bin/activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Set up environment variables

```bash
cp .env.example .env
```

Update `.env` with your PostgreSQL credentials:

```env
DATABASE_URL=postgresql://postgres:yourpassword@localhost:5432/churn_db
```

### 5. (Optional) Load raw data into PostgreSQL

```bash
python src/load_data.py
```

This reads the raw CSV, cleans column names, and pushes it to the `customers` table.

### 6. Run the API

```bash
uvicorn api.main:app --reload
```

API is live at `http://localhost:8000` — visit `http://localhost:8000/docs` for the interactive Swagger UI.

## API Endpoints

### `GET /`
Health check.

```json
{
  "status": "ok",
  "message": "ChurnShield API is running"
}
```

### `POST /predict`
Predict churn for a customer. See [Input Schema](#input-schema) below.

### `GET /docs`
Interactive Swagger UI — auto-generated by FastAPI.

## Input Schema

Send a JSON body with the following fields:

| Field | Type | Required | Description |
|---|---|---|---|
| `tenure` | float | ✅ | Months as a customer |
| `MonthlyCharges` | float | ✅ | Monthly bill amount |
| `TotalCharges` | float | ✅ | Total amount charged |
| `SeniorCitizen` | int (0/1) | ✅ | Is senior citizen |
| `Partner` | int (0/1) | ✅ | Has partner |
| `Dependents` | int (0/1) | ✅ | Has dependents |
| `PhoneService` | int (0/1) | ✅ | Has phone service |
| `PaperlessBilling` | int (0/1) | ✅ | Uses paperless billing |
| `charges_per_tenure` | float | ✅ | MonthlyCharges / tenure |
| `senior_high_charges` | int (0/1) | ➖ | Senior citizen with high charges (default 0) |
| `gender_Male` | int (0/1) | ➖ | Gender = Male (default 0) |
| `MultipleLines_Yes` | int (0/1) | ➖ | Has multiple lines |
| `InternetService_Fiber_optic` | int (0/1) | ➖ | Internet = Fiber optic |
| `InternetService_No` | int (0/1) | ➖ | No internet service |
| `OnlineSecurity_Yes` | int (0/1) | ➖ | Has online security |
| `OnlineBackup_Yes` | int (0/1) | ➖ | Has online backup |
| `DeviceProtection_Yes` | int (0/1) | ➖ | Has device protection |
| `TechSupport_Yes` | int (0/1) | ➖ | Has tech support |
| `StreamingTV_Yes` | int (0/1) | ➖ | Streams TV |
| `StreamingMovies_Yes` | int (0/1) | ➖ | Streams movies |
| `Contract_One_year` | int (0/1) | ➖ | Contract = 1 year |
| `Contract_Two_year` | int (0/1) | ➖ | Contract = 2 years |
| `PaymentMethod_Credit_card_automatic` | int (0/1) | ➖ | Pays by credit card |
| `PaymentMethod_Electronic_check` | int (0/1) | ➖ | Pays by e-check |
| `PaymentMethod_Mailed_check` | int (0/1) | ➖ | Pays by mailed check |
| `tenure_group_mid` | int (0/1) | ➖ | Tenure group: mid |
| `tenure_group_loyal` | int (0/1) | ➖ | Tenure group: loyal |

## Example Request & Response

**Request:**
```bash
curl -X POST "http://localhost:8000/predict" \
  -H "Content-Type: application/json" \
  -d '{
    "tenure": 3,
    "MonthlyCharges": 85.5,
    "TotalCharges": 256.5,
    "SeniorCitizen": 0,
    "Partner": 0,
    "Dependents": 0,
    "PhoneService": 1,
    "PaperlessBilling": 1,
    "charges_per_tenure": 28.5,
    "InternetService_Fiber_optic": 1,
    "PaymentMethod_Electronic_check": 1
  }'
```

**Response:**
```json
{
  "churn_prediction": 1,
  "churn_probability": 0.823,
  "risk_level": "high"
}
```
